together

  • I learnt about responsive images. We need Responsive images as:
    1. it allows Auto-resize based on screen size
    2. better performances to load higher resolution images only when needed
    3. provide better design control depending on device
  • I learnt how to use Github and how to sync changes to it.

    • Properties

    1. scale up and down -> width:100%;
    2. images to scale never larger than original size -> max-width:100%;
      (sets a maximum width for an element, which does not allow the width of that element to be larger than its max-width value)
    3. different resolutions -> use srcset
    4. fit images to a box -> background-size: cover;(Auto fit to box)
