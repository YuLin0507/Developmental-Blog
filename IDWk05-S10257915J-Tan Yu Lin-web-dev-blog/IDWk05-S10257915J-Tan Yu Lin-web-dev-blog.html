Objects, Arrays and more

  • (how to write an object) -> var object = {};
    eg.var bird={quote:'Nevermore'}
  • (retrieving data from the object) -> bird.quote / bird["quote"]
    output -> 'Nevermore'
  • (adding on properties to objects) -> bird.color = 'black'
    output -> var bird={quote:'Nevermore', color: 'black'}

Functions

  • A JavaScript function is executed when "something" invokes it (calls it).
  • use function and return
  • Function parameters are listed inside the parentheses () in the function definition.
  • Function arguments are the values received by the function when it is invoked.
  • Inside the function, the arguments (the parameters) behave as local variables.
  • rest parameters: three dots(...)in the parenthesis of a functions declaration
  • result of rest parameters is an array of arguments
